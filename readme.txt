PatronesArquitectónicos 
Tiffany Valeria Runza Quintero

En este proyecto JavaScript se ha hecho uso del Patrón arquitectónico Modelo-Vista-Controlador.
Se ha diseñado un proyecto que toma tareas, las guarda y las va a agregando a un costado para formar una lista.
En dicha lista se almcenan todas las tareas escritas en esa misma función. 
La lista marca en blanco si aún no se cumplen las tareas.
Al hacer click sobre cualquier tarea en la lista, el marcador se pondrá de verde. 
El color verde identifica que la tarea ya se ha hecho.
